ARIA View

ARIA View is a bookmarklet that reformats an HTML page to approximate how a page is perceived by assistive technology. If you're a developer, this is a great opportunity to get involved in an open source project during the embryonic stage. If you're looking a polished tools that will help you make your web site more accessible, check out HTML Codesniffer.

To use the bookmarklet, drag this link up to your bookmarks bar: Aria View
